# terraform-aws-fargate-threetier

## AWS 3tier application with below resources;
    ** Fargate
    ** Wordpress
    ** MySQL

### Prerequisities:
* Amazon AMI bastion
* Terraform v0.12.19
+ provider.aws v3.37.0
+ provider.random v3.1.0

## How to use

### `terraform init`

Prepare your working directory.

### `terraform plan`

Generate an execution plan.

### `terraform apply`
